Air Quality Industrial Platform
A desktop-based industrial AQMS parameter management application built with C# and WinForms.

This project is designed for industrial monitoring environments where operators need fast, reliable access to configuration and threshold parameters such as status, unit, min/max values, category, and severity.

The goal of the application is to provide a SCADA-style, operator-friendly interface with strong visual clarity, maintainable code, and a practical workflow for control-room and BMS-like environments.

Overview
Industrial monitoring systems often deal with dense parameter lists, where readability and quick decision-making are critical.

This application focuses on:

clear information hierarchy
fast visual feedback
severity-based highlighting
maintainable WinForms structure
reusable UI components
an industrial-style user experience
It is suitable for scenarios where operators must inspect, filter, and manage air quality monitoring parameters in a clean and efficient desktop interface.
